Local Helps

LocalHelps provides users with a simple, one-stop-shop to find local non-profit organizations or charities that accept their specific donations. We used JavaScript and a bit of Python for the back-end, and HTML and CSS for the UI/front-end. We created a csv file, which we then parsed through and created JSON objects of. From there, we were able to create our database of charities. We also integrated a Google Maps API for the map feature so that users can visualize where their local charities and non-profit organizations are.
